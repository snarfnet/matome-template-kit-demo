Matome Template Kit
===================

まとめサイトのURLを登録して、そのサイトの記事リンク一覧を作る買い切りテンプレートです。
WordPressは不要です。index.html をブラウザで開くだけで動きます。

内容
----
- index.html
- styles.css
- app.js
- README.txt

主な機能
--------
- まとめサイトURLの登録
- 同じサイト内の記事リンク候補を自動抽出
- 登録済みサイトの自動取得
- 全サイト更新
- 候補記事の一括追加
- 記事URLの手動追加
- 記事タイトル、サイト名、カテゴリ、説明文の編集
- 10種類のテンプレート切り替え
- 広告スペース管理
- Google広告で使いやすいサイズプリセット
- 公開ページプレビュー
- 人気リンク、カテゴリ、登録まとめサイト表示
- JSONバックアップ
- 公開HTMLコピー
- ローカル保存

使い方
------
1. index.html を開きます。
2. 「サイト登録」でまとめサイトのURLを入力します。
3. 「記事一覧を取得」を押します。
4. 表示された候補から、追加したい記事にチェックを入れます。
5. 「選択を追加」を押します。
6. 登録済みサイトは「全サイト更新」で新着記事を取得できます。
7. 管理画面を開いている間は、15分ごとに自動取得します。
8. 「公開ページ」で表示を確認します。
9. 必要なら「記事管理」でタイトルやカテゴリを編集します。
10. 「広告管理」で広告スペースを追加します。
11. 「書き出し」でJSONまたはHTMLを保存します。

自動取得について
----------------
登録済みサイトのうち「自動取得ON」のサイトだけが対象です。
管理画面を開いた時と、開いている間の15分ごとに新着記事を探します。
取得件数は10、20、30、50、100、300件から選べます。
100件以上は相手サイトや通信状況によって時間がかかる場合があります。
完全なサーバー常駐型ではないため、ブラウザを閉じている間は取得しません。
本格的に常時巡回したい場合は、サーバー版やCloudflare Workers版への拡張が必要です。

AdSenseメモ
-----------
Google AdSenseのディスプレイ広告はレスポンシブが基本です。
固定サイズを使う場合は、スマホ画面で横にはみ出さない場所に置いてください。
広告コード欄には、AdSenseの広告コードや仮バナーHTMLを貼れます。

取得について
------------
このテンプレートは、外部サイトを読みやすいテキストに変換する公開プロキシを使ってリンク候補を探します。
相手サイトの設定、アクセス制限、HTML構造によっては取得できない場合があります。
その場合は「記事管理」から手動でURLを追加してください。

注意
----
このテンプレートは、まとめサイトの記事リンクを紹介するリンク集を作るためのものです。
他サイトの記事本文を丸ごと転載する使い方は避けてください。
引用元リンクを残し、説明文は短い紹介や自分のコメントにしてください。
画像を使う場合は、利用できる画像か確認してください。

BOOTH販売文サンプル
-------------------
WordPressなしで、まとめサイトを集めたリンク集サイトを作れるテンプレートキットです。
まとめサイトURLを入力すると、記事リンク候補を抽出して一覧化できます。
管理画面、公開ページ、10種類のデザイン、広告枠管理、JSONバックアップ、HTML書き出しを収録。
買い切りなので、個人サイトや検証用サイトにすぐ使えます。
